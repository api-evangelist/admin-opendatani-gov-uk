# OpenDataNI (Northern Ireland) (admin-opendatani-gov-uk)

OpenDataNI (Northern Ireland) is a regional government open-data portal for United Kingdom running **CKAN**, exposing a standardized catalog API over ~1,066 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/admin-opendatani-gov-uk/refs/heads/main/apis.yml)
